# angularjsdemos
关于angular的实践

这是一个简单的关于angular路由、http服务配合node请求json数据的实例

node服务器返回json数据

通过angular内置服务$http请求数据

运行
首先开启服务cmd命令行 node index

浏览器打开index.html(由于是跨域,在index.js中已做允许所有请求的设置,如果报XMLRequest错误,可以改一下浏览器属性设置)
